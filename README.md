# Jeu Tic-Tac-Toe

Projet personnel en Python pour pratiquer :
- la logique conditionnelle
- les boucles
- la validation des entrées
- la structuration d’un programme

## Fonctionnalités

- Jouable à 2 joueurs
- Plateau 3x3 affiché dans la console
- Détection de victoire automatique
- Détection de match nul
- Validation des entrées pour éviter les erreurs

### Concepts Python utilisés

- Listes
- Boucles et conditions
- Fonctions
- Validation des entrées
- Docstrings pour documentation
